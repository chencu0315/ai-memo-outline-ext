# AI通用大模型问题目录面板 （谷歌浏览Chrome插件）

## 支持DeepSeek、DouBao、BaiDuYiYan等平台, 点击问题目录，可以快速跳转到对应的问题（锚点）

## 安装方法

### 打开 chrome://extensions/

### 启用开发者模式

### 点击“加载已解压的拓展程序”按钮， 导入插件所在的文件夹

### 在插件列表中，找到插件，则说明加载成功

### 打开相应模型网站，即可看到相应效果# ai-memo-outline-ext
